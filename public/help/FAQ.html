Frequently Asked Questions (FAQ)

Q. What features does PySchool.net provide?

  • Built with education in mind.
  • Runs entirely in a browser - no need to install/upgrade software.
  • Ability to save files in the cloud, so your files are safe and available anywhere.
  • Programming environment is available anywhere a web accessible browser is available.
  • Shareable Modules and scripts. An educator can create a module in which students can import and use.

Q. Are educators and students the only ones allowed to use PySchool.net?

  • Everyone can use PySchool.net. If you are interesting in learning more about Python you are welcome to use the site, free of charge.

Q. Is there a cost for PySchool.net?

  • A. Users will never be asked to pay for PySchool.net. One of our directives is to minimize barriers for learning programming. Asking for payment from users goes against this directive.

Q. There is no free lunch. Who pays for the site and developed resources?

  • A. Through fund raising efforts, such as our KickStarter campaign, we are raising funds to support the site as well as programming staff, to support the site, and add new features, etc.

Q. You say the site is free forever. How can you make such claims?

  • A. You make a good point. As they say, there are only two things guaranteed in life, death and taxes. Rightly, so. According to http://news.morningstar.com/pdfs/blanchett_lowbondyield_1301291.pdf unless there is a great economic catastrophe of some type (and with a little financial discipline), the fund should always contain sufficient funds. If for some reason, funds are depleted below an acceptable amount, there are some things that can be done to resolve the issue. One, additional donations, funds, etc can help bring the fund to a healthy amount, second, disbursement amounts can temporarily be reduced until the fund is brought back to healthy levels.

    For those interested in more details, a disbursement of 3% per year (approx. $60,000) will be authorized by the PySchool.net organization for maintenance, adding new features to the website, adding tutorials, and the day to day operations of PySchool.net

Q. How is Python code executed in a browser enviroment?

  • A. We use Brython. PySchool.net fully supports the development of Brython. See http://brython.info for more information.
